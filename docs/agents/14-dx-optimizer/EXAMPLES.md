# Examples: DX Optimizer

## Exemplo 1 — Configuração de Prettier

**Input:**
> Solicitação: Configurar Prettier para o projeto.

**Output:**

**[Configuração: .prettierrc]**
```json
{
  "singleQuote": true,
  "trailingComma": "all",
  "printWidth": 100
}
```

**[Comandos de Instalação]**
```bash
npm install --save-dev prettier
```

**[Instruções de Ativação]**
1. Instale a extensão do Prettier no VSCode.
2. Rode `npm run format` para formatar o projeto.
